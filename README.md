# Space-Invaders

**Игра реализована на языке C++ для консоли и для arduino**

# Arduino

Используется arduino Uno, экран Nokia 51110 Joystick Shield Arduino

**Подключение экрана:**

RST 12   (RESET)                          
CE  13   (CS)                             
DC  11   (Data/Command)                
DIN  10  (MOSI)                           
CLK  9   (SCK)                           
GND  GND
3.3V 7  (для удобства подключения экрана)

**Используемая библиотека**

https://github.com/adafruit/Adafruit-PCD8544-Nokia-5110-LCD-library

 

## Немного о структуре реализации и првилах

Игровое поле разбивается на три части (класса):

* Блок с вражескими кораблями
* Защитная стена
* Корабль, которым мы управляем

У классов BlockOfEnemies и MyShip есть функция Attack, выпускающая снаряд. Сам блок с вражескими кораблями меняет положение каждую секунду, при этом снаряды меняют положение в 4 раза быстрее. При попадании снаряда во вражеский корабль, тот погибает. При попадании снаряда в управляемый нами корабль, у нас отнимается жизнь. Когда количество жизней станет равно 0, игра заканчивается. 

Защитный блоки разрушаются после попадания в них 6 зарядов, не важно чьих, вражеского корабля, или нашего.

После уничтожения всех вражеских кораблей, повялется новый такой же блок.

Нильзя допускать, чтобы блок с вражескими кораблями опустился на защитный блок, игра считается оконченной. Если все защитные блоки разрушены, то игра считается оконченной при снижении вражеских кораблей до уровня нашего корабля.


## Как играть

В консоле управление произвоться с помощью клавишь a, w, d; движение влево, выстрел, движение вправо соответсвенно
В arduino с помощь кнопок D, A, B на джостике; движение влево, выстрел, движение вправо соответсвенно
